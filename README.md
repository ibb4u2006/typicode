# Typicode App

## Getting Started

First, create env file and add firebase configurations information to .env.local

`cp .env.local.example .env.local`

## Install the dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

## Install Husky

```bash
npm install husky
# or
yarn install husky
# or
pnpm install husky
```

## Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
